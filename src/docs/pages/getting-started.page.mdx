# Getting Started

---

This package provides a collection of pre-designed React components to streamline the process of building user interfaces.

### Installation

To get started with `@react-hive/honey-layout`, you need to have Node.js and npm installed on your machine.

~~~bash
npm install @react-hive/honey-layout
~~~

### Usage

Once installed, you can import the components into your React application and start using them right away.
One of the key components included in this package is `HoneyBox`.
